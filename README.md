# 🍋 Little Lemon Booking Website

## 📌 Description
A responsive restaurant table reservation website developed using React.

## 🚀 Features
- Online table booking
- Date selection
- Time selection
- Guest count input
- Form validation
- Responsive design

## 🛠 Technologies Used
- React
- HTML
- CSS
- JavaScript

## ▶️ Installation

```bash
npm install
npm start
```
